隐私政策

VS Code for Android（以下称为“本软件”）非常重视用户个人信息及隐私的保护，为了能准确抓捕程序异常，以及提供软件运行环境.
本软件会按照本隐私权政策的规定使用和收集您的设备信息。但本软件将以高度的勤勉、审慎义务对待这些信息，除友盟统计所必须上传信息之外，不会泄露其余信息，仅在本地处理。
除本隐私权政策另有规定外，在未征得您事先许可的情况下，本软件不会将这些信息对外披露或向第三方提供。
您在同意本软件服务使用协议之时，即视为您已经同意本隐私权政策全部内容。本隐私权政策属于本软件使用协议不可分割的一部分。

一、适用范围

1、当您打开本软件，创建文件时，您同意本软件使用存储权限；

二、信息安全

1、本软件除应用崩溃时收集设备信息写入日志，其它地方不再收集设备信息；
2、在使用本软件时，请您不要在非安全环境下运行，否则可能导致个人所编写的附加包源代码被他人窃取或删除等风险。

三、信息使用

1、本软件不会向任何无关第三方提供、出售、出租、分享或交易您的设备信息。
2、本软件亦不允许任何第三方以任何手段收集、编辑、出售或者无偿传播您的设备信息。任何本软件平台用户如从事上述活动，一经发现，本软件有权立即终止与该用户的服送有关其产品和服务。

四、信息披露

由于本软件仅涉及收集设备信息，不涉及收集个人信息，所以不会出现任何信息披露问题。

五、信息存储和交换

本软件收集的您的设备信息仅作为程序崩溃日志储存再本地，或统计平台收集的设备信息，用于区分不同手机用户。

六、本政策如何更新

1、为给您提供更好的服务，本软件会不断地对服务进行优化。未经您明确同意，我们不会削减您依据本隐私政策所应享有的权利。我们会通过在本软件移动端上发出更新版本并在生效前通过适当方式提醒您相关内容的更新。
2、若您不同意修改后的隐私政策，您应立即停止使用我们的产品和/或服务。如果您继续使用我们的产品和/或服务，则视为您接受我们对本政策相关条款所做的修改。
